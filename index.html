How does it works?

The user enters a prompt or theme in the designated input area, providing context for the lyrics they want to create.

Upon clicking the "Generate" button, the app captures the input and triggers an API request to the On-Demand IO service.

The On-Demand IO API processes the request using its AI algorithms, analyzing the prompt to generate relevant lyrics.




Once the lyrics are generated, the API sends the response back to the web app.

The generated lyrics are displayed in the designated area of the app for the user to review.

The user can edit the displayed lyrics as needed and save or download them for future use.
